🚀 Key Features

A comprehensive IAM solution built with modern technologies for enterprise-grade identity management

User Lifecycle Management

Complete user onboarding, role assignment, and offboarding workflows with automated access provisioning and deprovisioning.

RBAC & Conditional Rules

Role-based access control with conditional rules based on department, location, employment type, and custom attributes.

Access Review Workflows

Interactive access review process with current vs. desired state comparison and manual adjustment capabilities.

SaaS Integration

Built-in connectors for Google Workspace, GitHub, and extensible framework for custom SaaS applications.

HR Roster Import

CSV-based bulk user import with validation, automated sync tasks, and real-time processing status.

Interactive Dashboard

Real-time dashboard with user statistics, compliance tracking, and comprehensive audit logging.

🛠️ Technology Stack

Built with modern, scalable technologies for enterprise deployment

FastAPI Backend

High-performance async API with automatic documentation and JWT authentication

PostgreSQL/SQLite

Flexible database support with SQLAlchemy ORM and automated migrations

Redis & Celery

Background task processing for sync operations and real-time updates

Docker Deployment

Containerized architecture with docker-compose for easy deployment

Modern Web UI

Responsive dashboard with real-time updates and intuitive workflows

Security First

JWT tokens, HTTPS, input validation, and comprehensive audit logging

⚡ Quick Start

Get the IAM Lifecycle Demo running locally in minutes

# Clone the repository git clone https://github.com/mdisturbed/iam-lifecycle-demo.git cd iam-lifecycle-demo # Using Docker (Recommended) docker-compose up -d # Or run locally cd backend ./start_dev_environment.sh # Access the dashboard open http://localhost:8000/dashboard/

🔐 Default Credentials

# Admin Access Username: admin Password: admin123 # Read-only Access Username: readonly Password: readonly123
